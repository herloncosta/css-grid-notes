# CSS Grid

## Grid

- Bidimensional
- Divisão de toda a página em linhas e colunas
- Colocar os elementos onde quiser nessa divisão

---

## Grid ou Flexbox

- **Grid:** Duas dimensões (colunas e linhas)
- **Flexbox:** Uma dimensão (ou coluna ou linha)
- Um complementa o trabalho do outro
